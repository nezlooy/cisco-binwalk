## Supported formats

* ASA (`.bin`)
* PKG (`.pkg`)
* CSP (`.csp`)

## Installing

```
git clone https://github.com/nezlooy/cisco-binwalk
cp cisco-binwalk/magic/* ~/.config/binwalk/magic/
cp cisco-binwalk/plugins/* ~/.config/binwalk/plugins/
```

## Requirements

```
sudo apt install python3-crypto gzip bzip2 tar
```
